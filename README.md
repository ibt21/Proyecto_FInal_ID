# Proyecto_FInal_ID

## Tetris Game

## Descripción
Juego de Tetris clásico con sistema de usuarios y puntuaciones.

## Mecánica principal
- Piezas tetrominós caen y deben alinearse
- Líneas completas se eliminan y dan puntos
- Velocidad aumenta con el puntaje

## Cómo jugar
- Flechas: Mover y rotar piezas
- Espacio: Dejar caer rápidamente
- C: Reservar pieza

## Tecnologías
- HTML5 Canvas
- JavaScript
- CSS3
- localStorage
